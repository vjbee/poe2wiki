# Run n Gun Witchhunter

Mercenary / [[Witchhunter]] crossbow build by Woolie. Primary endgame loop: pre-stack **Stormblast Bolts**, wait through boss "one-shot reduction" ([[boss-anti-burst]]) window, dodge roll to auto-detonate via Mirage Archer + Explosive Shot. Clear via [[Galvanic Shards]] + Herald of Thunder.

First Mercenary build documented in the wiki.

## Two single-target variants

- **Stormblast Bolts** (primary, recommended) — pre-stack + detonate model.
- **Shockburst Rounds** (deprecated by the guide; "OLD") — full-charge Plasma Blast, then spam Shockburst Rounds on shocked enemies.

## Core gameplay loop (Stormblast Bolts variant)

### Single target

1. Fire 4–5x [[Stormblast Bolts]] as boss prep (most bosses "yap" before they can be damaged).
2. **Wait ~3 seconds** — bypasses boss "one-shot damage reduction" ([[boss-anti-burst]]).
3. Dodge roll → triggers [[Mirage Archer]] (which fires Explosive Shot), detonating the bolt stack.
4. Spam Stormblast Bolts at close range until magazine empty.
5. Emergency Reload, continue spamming.

### Clear

1. Spam [[Galvanic Shards]].
2. Herald of Thunder does the rest.

## Weapon set configuration (mandatory for endgame)

The Mirage Archer auto-proc only works when weapon sets are configured. Setup:

1. Right-click the **"I" Roman numeral** above the crossbow in inventory to **lock it to both sets** — avoids needing a second crossbow.
2. For each skill gem, click ">" and uncheck the slot ("I" or "II") so the gem appears only in the intended set.
3. A separate keybind appears for the Set II Crossbow Shot — bind it to something comfortable (Woolie uses middle-mouse).

### Set assignments

| Set I | Set II |
| --- | --- |
| Galvanic Shards | Stormblast Bolts |
| Herald of Thunder | Mortar Cannon |
| | Attrition |

[[Mirage Archer]] gets allocated to Set II so dodge rolls produce the Stormblast Bolt detonation.

## Ascendancy order

[[Witchhunter]] node order:

1. **Zealous Inquisition** — leveling power; **unspec after the final Ascendancy Trial**.
2. **Obsessive Rituals**.
3. **Ceremonial Ablution**.

## Defensive plan

Sorcery Ward × Ceremonial Ablution combo: **30% of Evasion is treated as HP**. Tankiness scales linearly with evasion stacking. Hyrri's Ire is the chase-tier chest unique for further evasion scaling.

Skin of the Loyal is used as the leveling/budget chest with a max "%all ele res" roll, specifically to **offset Rondel of Fragility's downside** — letting the build still hit resistance cap while keeping Fragility's offensive benefit.

## Auras

- **Act 1**: Herald of Ice (paired with the freeze/shatter leveling combo).
- **Act 3 or +30 Spirit**: Attrition.
- **After Galvanic Shards swap**: Herald of Thunder (replaces Herald of Ice).
- **Interlude 3**: Mirage Archer (requires weapon sets configured).

## Anointment

- **Unbound Forces** (budget) or **Acceleration** (more expensive).
