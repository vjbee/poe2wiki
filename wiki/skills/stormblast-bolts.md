# Stormblast Bolts

Crossbow attack skill. Primary single-target skill in the [[Run n Gun Witchhunter]] build.

## Gameplay model

Stormblast Bolts are **pre-stacked** on a target — the build fires 4–5 in sequence — then **detonated** via a separate proc:

- **Manual proc (campaign / early endgame)**: Explosive Shot, then Mortar Cannon + Explosive Grenade once gem tiers allow.
- **Auto proc (endgame)**: [[Mirage Archer]] with Explosive Shot on a dedicated weapon set; a dodge roll triggers the Mirage Archer's shot, which detonates the stack.

## Gem tier unlock

- Tier 9 gem unlocks Stormblast Bolts.

## Support links (per [[Run n Gun Witchhunter]])

- Elemental Armament II
- Rapid Attacks II
- Concentrated Area
- Close Combat II — **requires standing close to enemy** for the damage bonus.
- Lightning Attunement

## Boss-anti-burst interaction

The Run n Gun build pre-stacks bolts, then **waits ~3 seconds** before dodge-detonating. The wait specifically targets the [[boss-anti-burst]] window — letting it taper before unloading prevents the initial damage reduction from wasting the stacked bolts.

This is the cleanest current documentation of a build deliberately timing around boss-anti-burst.

## 0.5 outlook

Build author Woolie predicts possible **minor numeric nerfs** to Stormblast Bolts for [[patch-0.5.0]], without structural changes. Speculation, not GGG-confirmed.
