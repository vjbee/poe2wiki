# Galvanic Shards

Crossbow attack skill. Primary clear skill in the [[Run n Gun Witchhunter]] build.

## Role

Pure pack clear. Spam Galvanic Shards; Herald of Thunder handles cleanup. Replaces the leveling Permafrost Bolts + Fragmentation Rounds combo once the build transitions to lightning damage.

## Support links (per [[Run n Gun Witchhunter]])

- Elemental Armament II
- Double Barrel II
- Fork
- Innervate
- Projectile Acceleration III

## Reload pain point

Crossbow reloads can be annoying. The build guide notes that **Lineage gems on Galvanic Shards solve this** — Lineage supports are mentioned as a reload-anti-friction layer for the clear skill specifically.

## Weapon set

In a configured weapon-set build, Galvanic Shards belongs to **Set I** alongside Herald of Thunder. Set II is reserved for the single-target chain ([[Stormblast Bolts]], Mortar Cannon, Attrition, [[Mirage Archer]]).
