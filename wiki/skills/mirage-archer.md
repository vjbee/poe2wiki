# Mirage Archer

Spirit-reserved skill that automatically fires the linked attack skill on the player's behalf. In the [[Run n Gun Witchhunter]] build, Mirage Archer is the **endgame automation layer** that converts a dodge roll into a Stormblast Bolts detonation.

## Activation

- Costs Spirit (specific reservation not given in source).
- Requires **130 total Spirit** in the documented build (after +30 from gear at Interlude Act 3).

## Build use (Run n Gun Witchhunter)

Mirage Archer is linked with **Explosive Shot** on the build's Set II crossbow. When the player dodge rolls, the Mirage Archer fires Explosive Shot, which detonates the pre-stacked [[Stormblast Bolts]] on a target. This decouples damage from the trigger — the player keeps moving and dodging while the Mirage handles the proc timing.

**Critical**: Mirage Archer **requires weapon sets to be configured correctly** for the auto-proc to work — the Mirage must live on the same set as the skill you want it firing.

### Support links
